# DevExpress 本地引用说明

## 📋 如何添加本地 DevExpress DLL 引用

### 方法1: 在 Visual Studio 中添加（推荐）

1. **打开项目**
   - 在 Visual Studio 中打开 `永利系统.csproj`

2. **添加引用**
   - 在解决方案资源管理器中，右键点击项目
   - 选择 "添加" → "引用"
   - 点击 "浏览" 按钮

3. **选择 DevExpress DLL**
   
   根据您安装的 DevExpress 版本，DLL 通常位于：
   ```
   C:\Program Files (x86)\DevExpress 23.2\Components\Bin\Framework\
   ```
   
   或者：
   ```
   C:\Program Files (x86)\DevExpress XX.X\Components\Bin\Framework\
   ```

4. **需要添加的 DLL**
   
   最少需要以下 DLL（根据实际使用的控件）：
   
   - ✅ `DevExpress.Data.v23.2.dll` - 核心数据处理
   - ✅ `DevExpress.Utils.v23.2.dll` - 工具类
   - ✅ `DevExpress.XtraBars.v23.2.dll` - Ribbon 控件
   - ✅ `DevExpress.XtraEditors.v23.2.dll` - 编辑器控件
   - ✅ `DevExpress.XtraGrid.v23.2.dll` - Grid 控件
   - ✅ `DevExpress.XtraLayout.v23.2.dll` - 布局控件
   - ✅ `DevExpress.Dashboard.v23.2.Core.dll` - Dashboard 核心
   - ✅ `DevExpress.Dashboard.v23.2.Win.dll` - Dashboard WinForms

   **注意**: 版本号 `v23.2` 根据您实际安装的版本调整

5. **保存并构建**
   - 点击"确定"保存引用
   - 按 `Ctrl+Shift+B` 构建项目

---

### 方法2: 使用 DevExpress Project Converter（最简单）

DevExpress 提供了项目转换工具：

1. **打开项目**
2. **在 DevExpress 菜单中**
   - 点击 "DevExpress" 菜单
   - 选择 "Project Converter"
3. **选择版本**
   - 选择您安装的 DevExpress 版本
   - 工具会自动添加所有需要的引用

---

### 方法3: 直接在设计器中使用

当您在 Visual Studio 设计器中拖放 DevExpress 控件时：
- Visual Studio 会**自动添加**相应的 DLL 引用
- 这是最简单的方式！

**步骤：**
1. 打开 `Views/Main.cs` 设计器
2. 从工具箱拖放 DevExpress 控件
3. Visual Studio 自动添加引用
4. 保存即可

## 🔍 检查引用是否正确

### 在 Visual Studio 中查看
1. 展开项目的 "依赖项" → "程序集"
2. 应该能看到所有 DevExpress DLL

### 在 .csproj 文件中查看
引用添加后，.csproj 文件会自动更新，类似：

```xml
<ItemGroup>
  <Reference Include="DevExpress.Data.v23.2">
    <HintPath>C:\Program Files (x86)\DevExpress 23.2\Components\Bin\Framework\DevExpress.Data.v23.2.dll</HintPath>
  </Reference>
  <Reference Include="DevExpress.Utils.v23.2">
    <HintPath>C:\Program Files (x86)\DevExpress 23.2\Components\Bin\Framework\DevExpress.Utils.v23.2.dll</HintPath>
  </Reference>
  <!-- 其他引用... -->
</ItemGroup>
```

## 💡 常见问题

### Q1: 找不到 DevExpress DLL
**A**: 确认 DevExpress 已正确安装
- 检查 "控制面板" → "程序和功能"
- 查找 DevExpress 相关程序

### Q2: 版本号不匹配
**A**: 查看您安装的 DevExpress 版本
- 打开 DevExpress 安装目录
- 文件夹名称会显示版本号（如 DevExpress 23.2）

### Q3: 工具箱中没有 DevExpress 控件
**A**: 重新注册工具箱
- 在 Visual Studio 中
- "工具" → "选择工具箱项"
- 浏览到 DevExpress DLL 并添加
