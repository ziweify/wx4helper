## 🔧 使用指南

### 对于开发者

#### 查看文档
```bash
# 主文档
cat Unit.la/使用手册.md

# 快速参考
cat AI工作规则/快速参考.md

# Web库参考
cat Unit.la/Web库-快速参考.md
```

#### 更新文档
1. 修改代码
2. 打开 `Unit.la/使用手册.md`
3. 按规则更新相关章节
4. 更新"更新日志"
5. 与代码一起提交

### 对于 AI 助手

#### 工作流程
1. **代码变更后**，检查是否影响公开API
2. **如果影响**，立即更新 `Unit.la/使用手册.md`
3. **参考** `AI工作规则/库功能文档维护规则.md` 的检查清单
4. **验证**文档更新的完整性
5. **告知用户**已同步更新文档

#### 快速检查
```
代码修改完成？
  ↓
是否修改了公开API？
  ├─ 是 → 必须更新文档
  │     ├─ 打开 Unit.la/使用手册.md
  │     ├─ 更新相关章节
  │     ├─ 更新更新日志
  │     └─ 验证完整性
  │
  └─ 否 → 无需更新文档
```
