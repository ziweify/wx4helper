================================================================================
                        微信安装程序说明
================================================================================

请将微信 4.1.0.21 版本的安装程序放置在此目录下，并命名为：

    WeChatWin4.1.0.21.exe

完整路径应为：
    <程序根目录>\tools\WeChatWin4.1.0.21.exe

如果安装程序存在，软件启动时会在检测到版本不匹配时提供自动安装功能。

================================================================================
注意事项：
================================================================================

1. 安装程序必须严格命名为 "WeChatWin4.1.0.21.exe"
2. 安装过程需要管理员权限
3. 安装程序应该是官方的微信 4.1.0.21 版本安装包
4. 安装完成后程序会自动检测新版本

================================================================================
获取安装程序：
================================================================================

请联系管理员获取微信 4.1.0.21 官方安装包，并放置到此目录。

================================================================================
