# 🚀 登录命令快速参考

## 📍 位置

**配置管理器 → 高级设置 → 命令面板 → 【登录】按钮**

---

## 🎯 三大使用场景

### 1️⃣ 测试登录功能
```
打开配置管理器 → 高级设置 → 点击【登录】
查看命令结果面板 → 确认发送成功
查看浏览器日志 → 确认登录成功
```

### 2️⃣ 更新账号密码
```
基本设置 → 修改账号/密码 → 保存配置
高级设置 → 点击【登录】
浏览器自动更新登录凭据
```

### 3️⃣ 手动触发登录
```
确认浏览器已启动并连接
高级设置 → 点击【登录】
浏览器立即执行登录
```

---

## 📊 命令格式

```json
{
  "command": "Login",
  "data": {
    "username": "kkk99",
    "password": "123456"
  }
}
```

## ❌ 常见错误

### 错误1：账号密码为空
```
❌ 错误:账号或密码为空，请先在【基本设置】中填写
```
**解决**：在【基本设置】中填写账号密码

---

### 错误2：未选择配置
```
❌ 错误:未选择配置
```
**解决**：在左侧配置列表中选择一个配置

---

### 错误3：浏览器未连接
```
❌ 登录命令失败: 浏览器未连接
```
**解决**：启动浏览器，等待连接成功

---

## 🔄 与自动登录的关系

| 功能 | 自动登录 | 登录命令按钮 |
|------|---------|------------|
| **触发时机** | 浏览器启动时自动 | 手动点击 |
| **数据来源** | VxMain 自动发送 | 手动触发发送 |
| **使用场景** | 正常使用 | 测试/更新/调试 |

**结论**：
- **正常使用**：VxMain 自动发送，无需手动
- **调试测试**：使用【登录】按钮测试
- **更新密码**：使用【登录】按钮更新

3. ✅ 登录失败时，使用【登录】按钮重试
4. ❌ 正常使用无需点击（VxMain 会自动发送）
