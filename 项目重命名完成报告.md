3. 将文件夹 `永利系统` 重命名为 `YongLiSystem`

- 将 `永利系统.csproj.user` 重命名为 `YongLiSystem.csproj.user`（如果存在）

### 步骤3：使用 Git 记录重命名

```bash

cd YongLiSystem

git mv "永利系统.csproj.user" "YongLiSystem.csproj.user"
cd ..
```
