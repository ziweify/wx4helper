## 📋 规则列表

### 1. 终端命令执行规则 ⭐⭐⭐
**文件**: `终端命令执行规则.md`  
**重要性**: 🔴 极高（长期反复出现的问题）  
**适用范围**: 所有涉及终端命令执行的操作

**核心要点**:
- ❌ 不要使用`run_terminal_cmd`处理中文路径
- ✅ 创建脚本文件让用户执行
- ✅ 提供多种执行方式
- ✅ 说明清楚为什么需要用户手动执行
- ⚠️ PowerShell脚本中的中文字符串使用Base64编码

**快速判断**:
```
命令/路径包含中文？
  是 → 创建脚本 + 让用户执行
  否 → 可以使用 run_terminal_cmd
```

1. **明确性** - 规则必须清晰明确，不产生歧义
